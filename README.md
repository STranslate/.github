# STranslate

STranslate 是一款功能强大的翻译软件，旨在为用户提供便捷、高效的翻译服务。

## 主要功能

- **多引擎支持**：支持多种翻译引擎，包括但不限于谷歌翻译、微软翻译、百度翻译等
- **快捷翻译**：提供快捷键操作，实现快速翻译
- **划词翻译**：选中文本即可翻译，提升工作效率
- **截图翻译**：支持截图识别翻译功能
- **插件扩展**：丰富的插件系统，支持自定义扩展功能
- **多语言支持**：支持多种语言之间的互译
- **历史记录**：自动保存翻译历史，方便查询
